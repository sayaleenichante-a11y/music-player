# 🎵 Music Player

A modern and accessible web-based music player built using HTML, CSS, and JavaScript. This project focuses on delivering a smooth user experience along with accessibility (a11y) support.

---

## 🚀 Features

* ▶️ Play / Pause music
* ⏭️ Next / Previous track
* 🔊 Volume control
* 🎚️ Progress bar (seek functionality)
* 📱 Responsive design
* ♿ Accessibility (a11y) support
* 🎨 Clean and minimal UI

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## ⚙️ Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/sayaleenichante-a11y/music-player.git
```

2. Navigate to the project folder:

```bash
cd music-player
```

3. Open `index.html` in your browser

---

## 💡 How It Works

* Uses the HTML `<audio>` element for music playback
* JavaScript controls playback, UI updates, and track switching
* CSS ensures styling and responsiveness
* Accessibility features improve usability for all users
